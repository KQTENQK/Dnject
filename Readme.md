# Dnject



Simple windows tool for injecting dynamic-link libraries (DLLs) into running processes.



&nbsp;> Warning: This tool is intended \*\*solely for educational purposes\*\*, testing, and debugging your own applications. Using it to inject code into third-party processes without explicit consent may violate local laws and ethical guidelines. The user bears full responsibility for any consequences.

&nbsp;> Warning: Tool usage always ends up in antimalware detection.

&nbsp;> Warning: Do not use injectors unless you know what you are doing.



## Cli

 Usage:

 `DnjectT.exe <processName> <dllPath>`
